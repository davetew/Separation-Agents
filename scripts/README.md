# Scripts

This directory contains utility scripts for running simulations, optimizations, and verification loops.

## Files

*   **`run_loop.py`**: A script designed to execute iterative optimization or simulation loops. It typically handles the orchestration of multiple agents to converge on a design or process parameter set.

*   **`verify_optimization.py`**: Used to verify the results of optimization runs, likely by re-simulating the optimal points found by the agents.
