# Cost & Economics (`cost`)

This module handles the economic and environmental assessment of the designed processes.

## Components

*   **`economics_agent.py`**: The main agent for orchestrating economic analysis.
*   **`tea.py`** (Techno-Economic Analysis):
    *   Calculates CAPEX (Capital Expenditure) and OPEX (Operating Expenditure).
    *   Estimates ROI, NPV, and production costs.
*   **`lca.py`** (Life Cycle Assessment):
    *   Estimates environmental impacts, carbon footprint, and energy consumption.
