# Unit Operations (`units`)

This module provides models for specific unit operations used in separation processes.

## Components

*   **`base.py`**: Defines the base classes and interfaces for all unit operations.
*   **`flotation.py`**: Models for froth flotation cells.
*   **`hydromet.py`**: Models for hydrometallurgical processes (leaching, precipitation).
*   **`comminution.py`**: Models for crushing and grinding equipment.
*   **`magnetic.py`**: Models for magnetic separation units.
*   **`thickener.py`**: Models for sedimentation and dewatering (thickeners).
*   **`cyclone.py`**: Models for hydrocyclones.
