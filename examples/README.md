# Examples

This directory contains example scripts and configuration files that demonstrate how to use the `sep_agents` package and its agentic workflows.

## Files

*   **`agent_workflow_demo.py`**: A comprehensive demonstration script that runs a complete agent workflow. It covers:
    *   Defining detailed simulations (feasibility sweeps, speciation checks).
    *   Running equilibrium calculations using `EquilibriumAgent`.
    *   Interfacing with the `Reaktoro` backend.
    *   Saving and displaying results.
    *   **Usage**: `python3 agent_workflow_demo.py`

*   **`steel_slag_minimal.yaml`**: A sample input configuration file representing a minimal steel slag processing scenario. This file is often used as a test case for parsing and simulation setup.
