# Notebooks

This directory contains Jupyter notebooks used for interactive development, debugging, and demonstrating the capabilities of the Separation Agents.

## Content

*   **`SepAgentDevelopment.ipynb`**: The primary scratchpad for developing and testing new agent features interactively.
*   **`CompressibleFlowAgent.ipynb`**: Prototypes for compressible flow related agents (likely related to gas handling).
*   **`IDAESMetalFuelCycle.ipynb`**: Notebooks exploring integrations with the IDAES framework for metal fuel cycle processes.
